# Template Library (`/library`)

## Deprecation Notice

The `/library` route has been deprecated and currently results in a 404 Not Found error, as demonstrated in the screenshot below.

The functionality previously housed under `/library` has been consolidated and moved to the `/templates` route to provide a better user experience for browsing and utilizing available test generation templates.

For your generated scripts, please refer to the `/history` route.

![Library 404 Error](../docs/assets/library.png)

## Available Replacements

- **Browse Templates:** Navigate to [https://kr-test-automator.lovable.app/templates](https://kr-test-automator.lovable.app/templates)
- **View History:** Navigate to [https://kr-test-automator.lovable.app/history](https://kr-test-automator.lovable.app/history)
